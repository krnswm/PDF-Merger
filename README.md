# PDFx - PDF Merger Tool

A simple Node.js + Express app that merges multiple PDFs into one.

## Features
- Upload multiple PDFs
- Merge them into a single file
- Download instantly

## Tech Stack
- Node.js
- Express
- Multer
- pdf-merger-js

## How to Run
```bash
git clone https://github.com/krnswm/PDF-Merger.git
cd PDF-Merger
npm install
node server.js
